# Задача 01. FizzBuzz

## Условие

Напишите программу, которая выводит числа от 1 до N (N вводится с клавиатуры), но с несколькими исключениями:

- если число делится на 3 — вместо числа выведите `Fizz`;
- - если число делится на 5 — выведите `Buzz`;
  - - если число делится и на 3, и на 5 — выведите `FizzBuzz`;
    - - в остальных случаях выведите само число.
     
      - Каждое значение выводится на новой строке.
     
      - ## Пример
     
      - Ввод: `15`
     
      - Вывод:
     
      - ```
        1
        2
        Fizz
        4
        Buzz
        Fizz
        7
        8
        Fizz
        Buzz
        11
        Fizz
        13
        14
        FizzBuzz
        ```

        ## Что тренируем

        Циклы `for`, условные операторы `if/else`, оператор взятия остатка `%`.
