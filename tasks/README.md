## Как запускать решения

Для компиляции используйте любой компилятор C++ (g++ или clang++):

```bash
g++ -std=c++17 -O2 -o solution solution.cpp
./solution
```

## Хотите добавить свою задачу?

Создайте папку вида `NN-название`, добавьте в неё файл `README.md` с условием и `solution.cpp` с решением, а затем впишите новую строку в таблицу выше.
